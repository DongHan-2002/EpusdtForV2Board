# EpusdtForV2Board

> 该程序是用于[V2Board](https://github.com/v2board/v2board)对接[Epusdt](https://github.com/v03413/bepusdt)的支付插件

### 使用方法：

1. 下载目录下的Epusdt.php
2. 将下载的文件放在：【V2Board程序目录】/app/Payments目录下
3. 进入V2Board后台->支付配置->添加支付方式->选择Epusdt->输入你搭建Epusdt时的信息即可
